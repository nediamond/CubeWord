# CubeWord
A python script which prints even-length words as cubes.

### cubeword.py
```
    CubeWord  
   /u     /r  
  / b    / o  
 /  e   /  W  
CubeWord   e  
u   o  r   b  
b   r  o   u  
e   droWebuC  
W  /   e  /  
o /    b /  
r/     u/  
droWebuC  
```
### spaced_cubeword.py
```
        C u b e W o r d

o   /         b   /
r /           u /
d r o W e b u C
```
